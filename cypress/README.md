# Cypress Automation (E2E) — Bhanu

This folder contains my Cypress E2E practice to demonstrate automation basics used in Junior QA roles.

## What’s included
- Cypress specs (2 stable E2E tests)
- Test plan + test cases
- Test summary
- Evidence screenshots

## Site Under Test
https://example.cypress.io (public Cypress demo site)

## How to run locally

From the repo root:

1) Install dependencies
npm install

2) Open Cypress UI (Runner)
npx cypress open

3) Run headless in CLI
npx cypress run
